# Lumen Pro 现役功能白皮书 (v4.0.6-pro)

Lumen Pro 是专门为 Obsidian 深度定制的**专业级智能代理工作流引擎**。本次更新（v4.0.6-pro）对产品进行了全面精简，移除了冗余和认知负担较重的传统设计，让用户聚焦于对话本身，提供无缝、流畅的笔记协同体验。

本文档将详细介绍当前版本的所有**现役核心功能**。

---

## 🏛️ 核心现役功能目录

### 1. 极简对话与本地推理 (Lean Agent)
* **默认开启**：Lumen Pro 现已默认开启极简智能体模式，去除了繁琐的引导和调试面板，使对话框清爽纯粹。
* **高响应性对话**：深度优化与大模型的本地流式交互（Streaming），确保超低延迟的响应反馈。
* **轻量感知**：AI 会自动根据您的当前交互上下文进行敏捷推理，消除繁重的中间状态认知负载。

### 2. 白板视觉自动化 (Canvas Intelligence)
Lumen Pro 允许将 Obsidian Canvas（白板）转化为无代码 AI 编程与思维推演环境。
* **协议标签（现役）**：
  - `#lumen/start`：定义白板流的起点。
  - `#lumen/input`（可带编号如 `#lumen/input/1`）：标记输入源内容（支持直接拖入的 `.md` 笔记节点）。
  - `#lumen/step`（可带编号如 `#lumen/step/1`）：标记工作流的执行步骤，AI 会自动按拓扑顺序串行/并行推演并生成后续节点。
  - `#lumen/output`：标记输出靶向节点，AI 生成的最终结果会直接覆写该节点，而不会衍生出新的白板卡片。
* **多模态算子**：
  - `#lumen/image`：原生支持在白板节点中自动生成图像，并自然落位于配置的附件文件夹中。
  - `#lumen/video`：支持调用硅基流动/Luma/MiniMax 接口在节点生成高品质短视频。
  - `#lumen/audio`：支持音频生成与播报。
  - `#lumen/vision`（旧别名 `#lumen/image2prompt`）：视觉提词算子，支持上传/引用图片并反向生成提示词。
* **连线推演与智能节点感应**：直接拖入 `.md` 笔记文件至 Canvas，Lumen 自动感应为有效输入；配合 AI 连线引擎，实现逻辑结构在白板上的自然延展。

### 3. 动态上下文引用 (Dynamic Context)
在对话中实现精准、类型化的信息传递，避免无效 Token 消耗。
* **`@` 引用指令**：
  - **引用笔记**：输入 `@笔记名称` 即可在当前对话上下文中精准注入特定笔记内容。
  - **引用属性表格 (Bases)**：输入 `@.base文件名` 智能解析表格定义及相关数据。
  - **引用文件夹**：输入 `@文件夹名` 限制对话引用的检索范围或为生成文件提供归档靶向。

### 4. 项目感知与智能路由 (Project-Aware Router)
移除了底部的浮动状态条，改用更自然的自动化命令路由方式：
* **斜杠指令定位**：使用 `/project@meeting`、`/project@decision` 等命令。
* **智能属性推断**：AI 根据当前操作文件的 YAML 属性、文件路径或 `@文件夹` 引用，自动智能匹配其所属的项目以及生成文件的目标路径。
* **落位规则**：生成的文件和图片会直接保存到 Obsidian 设置的默认附件目录或被路由的项目文件夹中，让笔记组织保持自然、纯净。

### 5. 双轨语义记忆库 (Dual-Track Local RAG)
真正懂你库内上下文的本地高维语义搜索。
* **本地向量搜索**：利用高性能嵌入模型对全库/指定文件夹笔记构建本地向量索引。
* **双轨检索机制**：混合被动向量 RAG 检索（70% 权重）与主动 Session 总结（30% 权重），大幅度提高复杂长上下文的检索精准度。
* **灵感召唤 (Summoning)**：在白板中能够通过语义相似度自动拉取并连线呈现与选定节点高度关联的库内笔记。

### 6. 多文件分发引擎 (Multi-File Planner)
* **批量生成**：支持通过在工作流步骤中配置 YAML 结构，一次性批量创建或拆分多篇 Markdown 笔记。
* **原子化协作**：避免将所有输出塞在一篇笔记中，实现模块化、原子化的知识归档。

### 7. Telegram 移动端异步同步
* **远程同步**：支持随时随地向 Telegram Bot 发送文字或语音。
* **AI 智能入库**：机器人自动将语音转译为文字，并通过 AI 进行排版和润色，自动同步到本地 Obsidian 笔记库中。

## 🗑️ 已停用/移除的功能说明

为确保您的 Obsidian 库回归纯净体验，降低学习成本，以下功能**已完全从当前版本中停用或移出界面**：
* ❌ **AutoResearch (科研调研算子)**：移除了网页深度检索及 ` #lumen/research ` 算子，回归纯对话和本地库检索。
* ❌ **OpenCLI (本地工具链算子)**：移除了 Obsidian 内部直接驱动本地终端脚本的 ` #lumen/opencli ` 算子，保证沙箱环境的绝对安全与简洁。
* ❌ **Floating ProjectStatusBar (底部浮动项目状态条)**：移除了对话框底部常驻的浮动项目/阶段配置条，改由更为轻量的斜杠指令与 `@` 机制取代。
* ❌ **ImageResultCard Star Button (打星功能)**：去除了生图卡片右下角的打星（☆）收集操作，生成的图片文件直接自动归档落位，不再引入项目阶段评级负载。
