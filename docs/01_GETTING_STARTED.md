# 01 Getting Started: Installation & Activation

Welcome to LUMEN Pro. This guide will help you install the plugin and unlock its professional agentic features.

### 📦 Installation Methods

#### Method 1: BRAT (Recommended for Beta/Updates)
Using the **BRAT** (Beta Reviewer's Auto-update Tool) plugin ensures you always have the latest version of LUMEN Pro.
1. Install **BRAT** from the Obsidian Community Plugins.
2. Go to **Settings > BRAT > Beta Plugin List**.
3. Click **Add Beta plugin**.
4. Enter the repository URL: `leoyang1984/lumen-public`
5. Click **Add Plugin**.
6. Enable **Lumen Pro** in your Community Plugins list.

#### Method 2: Manual Installation
1. Visit the [GitHub Releases](https://github.com/leoyang1984/lumen-public/releases) page.
2. Download `main.js`, `manifest.json`, and `styles.css`.
3. Locate your vault's plugin folder: `.obsidian/plugins/`.
4. Create a new folder named `lumen-pro`.
5. Move the downloaded files into `lumen-pro` and enable the plugin in Obsidian.

#### 2. Enter License Key
1. Find the **License Key** field in the settings.

# 01 快速上手：安装与激活

欢迎使用 LUMEN Pro。本指南将引导您完成插件安装并解锁专业的代理（Agentic）功能。

### 📦 安装方法

#### 方法 1：使用 BRAT 安装（推荐，方便后续更新）
通过 **BRAT** 插件安装可以确保您第一时间获得 LUMEN Pro 的版本更新。
1. 在 Obsidian 社区插件市场中安装 **Obsidian 42 - BRAT**。
2. 进入 **设置 > BRAT > Beta Plugin List**。
3. 点击 **Add Beta plugin** 按钮。
4. 输入仓库地址：`leoyang1984/lumen-public`
5. 点击 **Add Plugin**。
6. 在社区插件列表中开启 **Lumen Pro**。

#### 方法 2：手动安装
1. 前往 [GitHub Releases](https://github.com/leoyang1984/lumen-public/releases) 发布页。
2. 下载 `main.js`, `manifest.json`, `styles.css` 这三个文件。
3. 打开您的库目录中的插件文件夹：`.obsidian/plugins/`。
4. 新建一个名为 `lumen-pro` 的子文件夹。
5. 将下载的文件拷贝进去，并在 Obsidian 中开启插件。
